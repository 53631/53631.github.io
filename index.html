Lists

A list is a data structure (a way of organising data). A list has items of the same datatype and may include duplicates. The number of items in a list can grow or shrink. Items can be added into a particular position in a list.

One way to implement a list in Java is to use an ArrayList. A number of different operations can be performed on an ArrayList. These are shown in the table below.

The examples are based on the following list with the identifier myList: [45, 66, 41, 13, 19, 8]

List Operation Description Example Output
add(item) Appends the specified item to the end of a list. myList.add(1) --
add(index, item) Inserts the specified item at the specified position index in a list. myList.add(2, 1) --
clear() Removes all of the elements from this list. myList.clear() --
lastIndexOf(item) Returns the index in this list of the last occurrence of the specified element, or -1 if the list does not contain this element. myList.lastIndexOf(13) 3
remove(index) Removes the element at the specified position in this list. Shifts any subsequent elements to the left (subtracts one from their indices). myList.remove(1) --
get(index) Gets the element of a specified position within the list. myList.get(1) 66
size() Returns the number of elements in this list i.e the size of the list. myList.size() 6
contains(item) Returns true if this list contains the specified element. myList.contains(123) false
indexOf(item) Returns the index in this list of the first occurrence of the specified element, or -1 if the list does not contain this element. myList.indexOf(66) 1
